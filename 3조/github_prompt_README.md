# 국내 비대면진료 기반 당뇨병 관리의 효과성과 한계
## 문헌고찰 및 뉴스분석 기반 소논문·포스터 작성 가이드

> 학부생이 생성형 AI를 활용해 **소논문**과 **포스터 발표자료**를 쉽게 작성할 수 있도록 만든 실습형 프롬프트 모음입니다.

---

## 프로젝트 개요
이 자료는 다음 주제를 바탕으로 작성되었습니다.

**주제**  
국내 비대면진료 기반 당뇨병 관리의 효과성과 한계: 문헌고찰 및 뉴스분석

**연구 형태**  
- 문헌고찰(scoping review / 간이 systematic review 형태)
- 뉴스 내용분석
- 문헌과 뉴스의 통합 논의

**활용 결과물**  
- 소논문 초안 작성
- 포스터 발표자료 구성
- 발표 대본 작성
- 그래프/도표 아이디어 도출

---

## 이 저장소의 목적
이 저장소는 학생들이 아래 과정을 **복사-붙여넣기 중심**으로 수행할 수 있게 돕는 것을 목표로 합니다.

1. 주제와 연구문제 정하기  
2. 논문·뉴스 자료 수집하기  
3. 문헌 정리표와 뉴스 코딩표 만들기  
4. 생성형 AI로 소논문 초안 작성하기  
5. 포스터용 문장과 발표 대본 만들기  
6. 그래프와 표 설명 문장 만들기

## 학생용 주의사항

### 꼭 지켜야 할 원칙
- AI가 **없는 논문, 없는 뉴스, 없는 통계**를 만들지 않도록 주의합니다.
- 참고문헌은 **반드시 원문 확인 후** 직접 정리합니다.
- AI가 쓴 문장을 그대로 제출하지 말고, 사람이 다시 검토하고 다듬습니다.
- 결과와 논의를 구분해서 작성합니다.
- 내가 정리한 표와 자료를 넣은 뒤 AI를 사용합니다.

### 추천 작업 방식
- 먼저 엑셀로 **문헌 정리표**와 **뉴스 코딩표**를 만듭니다.
- 그 다음 프롬프트에 자료를 붙여 넣어 문단을 생성합니다.
- 마지막에 사람이 내용을 확인하고, 참고문헌과 표현을 정리합니다.
